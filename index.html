Dự án

Dự án Quản lý Thư viện

Phát triển ứng dụng quản lý thư viện hoàn chỉnh với khả năng quản lý sách, độc giả, và các giao dịch mượn/trả sách. Thiết kế giao diện người dùng thân thiện và dễ sử dụng.

Công nghệ: Java, MySQL, Tkinter

Dự án Ứng dụng Thời tiết

Xây dựng ứng dụng web responsive hiển thị thông tin thời tiết theo vị trí người dùng. Tích hợp API thời tiết và hiển thị dữ liệu trực quan với biểu đồ và animations.

Công nghệ: HTML5, CSS3, JavaScript, Weather API
